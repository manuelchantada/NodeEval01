Consigna

Diseñar un servidor NodeJS que pueda servir un sitio con 4 secciones, 3 de ellas serán estáticas y una cuarta deberá contener un formulario de contacto a través del cual se tomen los datos del usuario y se haga el envío de mail haciendo uso del paquete nodemailer.js.

El sitio tendrá que estar formadopor los html correspondientes a cada página, una hoja de estilos .css , o se puede usar bootstrap o algun otro framework de este tipo para los estilos, imágenes, y algún plugin de JQuery. El archivo js tendrá que incluir todos los módulos necesarios y el código correspondiente para poder ejecutarlo.
